# AdQuest - Рекламная CAPTCHA система

**AdQuest** - инновационная система защиты от ботов, которая монетизирует процесс верификации пользователей через показ рекламы. Пользователи просматривают рекламный контент и выполняют простые задания для подтверждения своей человечности.

## Обзор системы

AdQuest объединяет защиту от ботов с рекламной монетизацией, создавая win-win ситуацию для всех участников:

- **Владельцы сайтов** получают доход от каждой верификации
- **Рекламодатели** получают качественную аудиторию с гарантированным просмотром
- **Пользователи** видят релевантную рекламу вместо сложных капч

## Компоненты системы

### AdQuest Core
**Основной backend сервис (Rust + Node.js)**
- Высокопроизводительный Rust движок (Challenge Engine, Billing Engine)
- Управление рекламными кампаниями
- API для генерации и валидации заданий
- Система таргетинга и аналитики
- Защита от мошенничества
- Интеграция с ЕРИР

### AdQuest Widget
**JavaScript SDK для интеграции**
- Легкое встраивание на любой сайт
- Адаптивный дизайн для всех устройств
- Множество типов рекламных заданий
- Кастомизация внешнего вида
- Соответствие ФЗ-38 и ЕРИР

### AdQuest Admin
**Панель управления (React)**
- Интерфейс для рекламодателей
- Дашборд для владельцев сайтов
- Аналитика и отчетность
- Управление финансами

### AdQuest Docs
**Документация и гайды**
- API документация
- Примеры интеграции
- Лучшие практики
- Техническая поддержка

## Ключевые особенности

### Производительность
- Challenge Engine: <10ms p50 latency, >10,000 RPS
- Снижение памяти на 70% по сравнению с Node.js
- Снижение CPU на 67% по сравнению с Node.js
- Финансовая точность: 4 знака после запятой

### Соответствие законодательству
- 152-ФЗ "О персональных данных" - все данные в российской юрисдикции
- 242-ФЗ "О локализации данных" - персональные данные россиян только в РФ
- ЕРИР - полная интеграция с Единым реестром интернет-рекламы
- НДС - автоматический расчет и уплата НДС

### Модель монетизации CPV
**Распределение доходов:**
- 60% - владельцу сайта
- 35% - AdQuest (биллинг-центр + ОРД)
- 5% - антифрод фонд

## Быстрый старт

### Для владельцев сайтов

1. Зарегистрируйтесь на admin.adquest.ru
2. Добавьте ваш сайт и получите Site Key
3. Интегрируйте виджет на ваш сайт

```html
<script src="https://cdn.adquest.ru/widget/v1/adquest.min.js"></script>
<div id="adquest-captcha"></div>
<script>
  AdQuest.render('adquest-captcha', {
    siteKey: 'YOUR_SITE_KEY',
    callback: function(token) {
      // Отправить токен на сервер для верификации
      submitForm(token);
    }
  });
</script>
```

### Для рекламодателей

1. Зарегистрируйтесь на admin.adquest.ru
2. Создайте рекламную кампанию
3. Загрузите креативы (видео, баннеры, интерактив)
4. Настройте таргетинг и бюджет
5. Запустите кампанию и отслеживайте результаты
